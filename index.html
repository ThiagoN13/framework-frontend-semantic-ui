Instalar o Semantic UI
  • Você pode baixar usando o semantic UI do pacote NPM, mas é necessário que instale o GULP para compilar o arquivo

  • O Gulp roda no nodejs então é preciso que você o tenha instalado no seu computador, caso esteja tudo ok, basta rodar o comando abaixo na sua linha de comando para instalar o CLI:

    Gulp é um módulo NPM e deve ser instalado globalmente

                      
                    npm install -g gulp
  • Instale o semantic UI através do pacote NPM 
                      
                    
                      npm install semantic-ui --save
                      cd semantic/
                      gulp build
